Why AgentPet

Stop tab-hunting across terminals

Run several agents in parallel and let AgentPet surface who needs you.

Multi-agent monitor

A live menu bar list of every agent: its state, what it's doing, and a real-time timer.

At a glance

The menu bar icon shows how many agents run, and turns orange when one needs your input.

Desktop pet

A pet reacts to agent activity — working, waiting, done, celebrate — with optional chat bubbles.

Works with 6 agents

One-tap hook install for Claude Code, Cursor, Codex, Gemini, opencode & Windsurf. GLM works through Claude Code. Any CLI via agentpet run.

Notifications & sounds

Get alerted, and hear it, the moment an agent finishes or needs you.

Bring your own pet

Browse an online pet library, download in one click, and map animations to states.

Get started

Up and running in a minute

1

Install

Run this, or download the DMG.

brew install --cask ntd4996/tap/agentpet
2

Connect

Open Settings and one-tap install the hook for Claude Code, Cursor, Codex, Gemini, opencode or Windsurf.

3

Code

Run your agents. The menu bar and your pet now react to everything in real time.
