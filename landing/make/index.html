How to make a pet

A pet is one transparent PNG sprite sheet: each row is an animation, each column a frame, with transparent gaps between frames. You only upload the PNG, we generate the pet.json for you.

Make one with AI (ChatGPT)

Tested with ChatGPT, works with any image AI that can look at an attached picture and generate one. Do it in two steps so the AI locks onto the format first.

Step 1. Attach the sample sheet above to ChatGPT and have it study the layout:

Step 2. Attach a picture of your character (or just describe it) and ask for the sheet:

Download the PNG it makes, then drop it into Upload a sheet. The preview plays each animation, so you can check it before posting. If only one frame shows up, the background was not transparent: ask the AI for a transparent background, or clean it with remove.bg.

No image-to-image AI? Make single frames instead and let us stitch them: ask for separate frames, then use Build from frames.
